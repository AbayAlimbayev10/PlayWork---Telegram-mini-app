# PlayWork---Telegram-mini-app
PlayWork is an AI-powered, game-based system for developing human potential through real-life actions. It transforms work, personal growth, and performance into a structured game where progress is measured, guided, and continuously optimized by artificial intelligence.  
PlayWork is a Telegram-first daily challenge system for companies.

It helps employees build consistency, focus and energy through simple daily challenges.

MVP scope:
- Company onboarding
- Daily challenges (1 per day)
- Completion tracking
- Streaks and points
- Weekly company summary
